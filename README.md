# kommunpolitiken – protokollarkiv

Arkiv med kommunala protokoll (offentliga handlingar) som ligger till grund för analyserna på [kommunpolitiken.se](https://kommunpolitiken.se).

Protokollen är hämtade från kommunernas egna publiceringskanaler:

- **Boden** – diariet (arende-handling.boden.se); PDF:erna ligger i `<år>/`
- **Luleå** – lulea.se; PDF:erna ligger i `lulea/<år>/`
- **Pajala** – pajala.se (anslagstavlan); PDF:erna ligger i `pajala/<år>/`

Filerna serveras till sajtens "läs protokollet"-länkar via detta repo. Dokumenten är allmänna handlingar enligt offentlighetsprincipen; upphovsrätten till innehållet ligger hos respektive kommun i den mån sådan finns.

Frågor? Se [kommunpolitiken.se/om](https://kommunpolitiken.se/om) eller mejla tipsa@kommunpolitiken.se.
